Class: Vehicle

# Vehicle

The Vehicle class represents a vehicle within the game. Every vehicle in the game is an instance of this class. The Vehicle's task is to transport passengers to their desided destinations.

Vehicles can travel between Cities when these cities are connected by Roads.

Vehicles are able to carry Passengers up to a max capacity, which depends on their type.

Example

Example of a vehicle going back and forth between cities A, B and C

function init (vehicles, cities, passengers, game) {

   let vehicle = vehicles[0];
   vehicle.moveTo(['B', 'C']);

   vehicle.on('visitCity', function(city) {
     if (city.id == 'C')
       vehicle.moveTo(['B', 'A']);
     else if (city.id == 'C')
       vehicle.moveTo(['B', 'A']);
   });
}

   });
