Protótipo da Documentação

Neste apêndice é feito um protótipo da documentação. A documentação é o documento que é destinado a auxiliar o jogador durante o jogo listando os objetos como veículos, classes e seus atributos.

API - Classes

Classe Vehicle

Propriedade

Tipo

Explicação

moveTo(targetCity || targetRoute)

método

Responsável por mover o veículo para uma cidade ou para várias cidades, através de uma rota.

load([passenger])

método

Responsável por carregar passageiros no veículo, parâmetro opcional define explicitamente qual passageiro deve ser carregado.

unload(passenger)

método

Descarrega um passageiro na cidade atual do veículo.

passengers

Array

Lista contendo os passageiros atuais dentro do veículo.

passengersCapacity

Inteiro

Quantidade máxima de passageiros que este veículo suporta.

lastCity

City

Última cidade que o veículo passou

currentRoute

Array

Lista de cidades (City) contendo a rota de destino deste veículo.

on(’idle’, function)

Evento

Executado toda vez que o veículo se encontra sem nenhuma ação a ser feita (quanto não houver mais nenhuma cidade a ser visitada na sua lista de rotas).

on(’loadPassenger’, function)

Evento

Executado toda vez que o veículo carregar um passageiro. A função callback recebe como parâmetro o passageiro.

on(’visitCity’, function)

Evento

Executado toda vez que o veículo estiver passando por uma cidade. A função callback recebe como parâmetro a cidade.
